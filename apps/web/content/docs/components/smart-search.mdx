      }

      return (

          placeholder="Search documentation..."
          searchHistory={true}
          suggestions={suggestions}
          searchFilters={searchFilters}
          onSearch={handleSearch}
          onSuggestionSelect={(suggestion) => console.log("Selected:", suggestion)}
          resultCount={searchResults.length}
          debounceMs={300}
          urlSync={true}
        />
      )

## Installation

Install the Smart Search component using your preferred package manager.

```npm
npx shadcn@latest add https://rigidui.com/r/smart-search
````

## Features

The Smart Search component is packed with powerful features designed to provide an exceptional user experience.

<Cards>
  <Card title="Real-time Search">
    Debounced search input with real-time results and visual feedback for search
    states.
  </Card>
  <Card title="Search History">
    Optional search history with local storage persistence and easy access to
    recent searches.
  </Card>
  <Card title="Smart Filters">
    Configurable filter chips and advanced query parsing with filter:value
    syntax support.
  </Card>
  <Card title="Keyboard Navigation">
    Full keyboard support with arrow keys for navigation, Enter to select, and
    Ctrl+/ to focus.
  </Card>
</Cards>
