## Installation

Install the Multi-Step Form Wrapper component using your preferred package manager.

```npm
npx shadcn@latest add https://rigidui.com/r/multi-step-form-wrapper
```

## Features

The Multi-Step Form Wrapper component is packed with powerful features designed to provide an exceptional user experience.

<Cards>
  <Card title="Step-by-Step Form Flow">
    Guide users through complex forms with a clear step-by-step interface and
    visual progress tracking.
  </Card>
  <Card title="Advanced Validation">
    Comprehensive step-by-step validation with Zod schemas, custom error
    messages, and validation error callbacks.
  </Card>
  <Card title="Auto-Save & Persistence">
    Automatically save form progress to localStorage with configurable
    debouncing and persistent form state.
  </Card>
  <Card title="Smooth Animations">
    Beautiful step transitions with configurable animation duration and seamless
    user experience.
  </Card>
  <Card title="Flexible Configuration">
    Highly customizable with step lifecycle hooks, optional steps, progress
    bars, and accessibility features.
  </Card>
</Cards>
