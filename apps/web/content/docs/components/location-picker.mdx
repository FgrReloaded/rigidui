## Features


<p className="md:text-sm mb-4!  text-gray-700 dark:text-gray-300 max-w-3xl">
  The Location Picker component is packed with powerful features designed to
  provide an exceptional user experience.
</p>

<Cards>
  <Card title="Location Search">
    Users can search for locations by name, address, or postal code with
    real-time suggestions.
  </Card>
  <Card title="Dual Variants">
    Choose between popover variant for compact layouts or inline variant for
    full-featured forms.
  </Card>
  <Card title="Autocomplete Suggestions">

    with debounced search.

    API.
  </Card>
