## Installation

<p className="md:text-sm  text-gray-700 dark:text-gray-300 max-w-3xl">
  Install the Location Picker component for your platform using your preferred
  package manager.
</p>

<Tabs items={["Web", "React Native"]}>
  <Tab value="Web">
    ```npm
    npx shadcn@latest add @rigidui/location-picker
    ```
  </Tab>
  <Tab value="React Native">
    ### Prerequisites [!toc]

    First, ensure you have React Native Reusables properly set up in your project. Follow the complete setup guide at [React Native Reusables Installation](https://reactnativereusables.com/docs/installation).

    ### Install Dependencies [!toc]

    Install the required Expo dependencies for location functionality:

    ```bash
    npx expo install expo-device expo-location
    ```

    ### Configure Expo Plugins [!toc]

    Add the `expo-location` plugin to your `app.json` file:

    ```json title="app.json"
    {
      "expo": {
        "plugins": ["expo-location"]
      }
    }
    ```

    For additional configuration options and permissions setup, refer to the [Expo Location documentation](https://docs.expo.dev/versions/latest/sdk/location/).

    ### Install Component [!toc]

    ```npm
    npx shadcn@latest add @rigidui/location-picker-rn
    ```

  </Tab>
</Tabs>

## Features

<p className="md:text-sm mb-4!  text-gray-700 dark:text-gray-300 max-w-3xl">

<Cards>
  <Card title="Location Search">
    Users can search for locations by name, address, or postal code with
    real-time suggestions.
  </Card>
  <Card title="Dual Variants">
    Choose between popover variant for compact layouts or inline variant for
    full-featured forms.
  </Card>
  <Card title="Autocomplete Suggestions">
    As users type, the component provides autocomplete suggestions for locations
    with debounced search.
  </Card>
  <Card title="Cross-Platform Support">
    Available for both Web and React Native with platform-specific optimizations
    and native geolocation support.
  </Card>
  <Card title="Geolocation Support">
    Allows users to quickly use their current location with browser geolocation
    API on Web and native location services on React Native.
  </Card>
</Cards>
