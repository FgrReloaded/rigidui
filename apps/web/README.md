# RigidUI Website

The official website and documentation for RigidUI components.

## Development

This website is part of the RigidUI monorepo.

### Getting Started

From the root of the monorepo:

```bash
# Install dependencies
pnpm install

# Start development server
pnpm dev

# Build for production
pnpm build
```
